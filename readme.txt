Simple Instagram plugin for WordPress that allows an developer to Share Sites Instagram Photos using "Widgets". 

== Description ==

Easy to use Instagram Sharing widget ( Share Your Instagram's With The World )

Example link able to share: https://www.instagram.com/p/BXqAOSXDb38


== Installation ==

1. Upload `1337-Instagram.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

or

login to wp-admin
- Dashboard > Plugins > Add New
